# Secure Freelance Marketplace

Платформа для безпечної організації freelance-роботи між користувачами.

**Стек:** FastAPI · PostgreSQL · SQLAlchemy · Alembic · AES-GCM · RSA-OAEP · RSA-PSS · SHA-256

## Quick Start

```bash
cp .env.example .env
docker compose up --build
```

API docs: http://localhost:8000/docs

## Project Structure

```
backend/app/
  main.py

  core/               — config, exceptions, logging, middleware
  database/           — SQLAlchemy base, session, transactions
  common/             — enums, pagination, responses, validators, types

  modules/
    accounts/         — registration, login, JWT, RBAC, RSA key records
    marketplace/      — tasks, offers, search
    contracts/        — contract lifecycle, state machine
    communication/    — encrypted REST chat
    delivery/         — encrypted chunk-based file upload
    payments/         — simulated escrow, ledger
    moderation/       — disputes, moderator decisions
    security/         — AES-GCM, RSA-OAEP, RSA-PSS, SHA-256, key management
    audit/            — audit log, signed critical events

alembic/              — database migrations
docs/                 — project documentation
security/reports/     — SAST & audit reports (generated by CI)
storage/              — encrypted file chunks (local, not committed)
```

## CI/CD

| Job | Trigger | What it does |
|---|---|---|
| lint | push/PR → develop, main | flake8 |
| test | after lint | pytest + PostgreSQL service |
| sast | after test | Bandit, fails on HIGH severity |
| deploy-staging | push → develop | deploy stub |

## Security

- AES-GCM для шифрування messages і file chunks
- RSA-OAEP для key wrapping (окрема RSA encryption key pair)
- RSA-PSS для підпису critical events (окрема RSA signing key pair)
- SHA-256 для file fingerprint і payload hashes
- JWT access/refresh tokens · Argon2 password hashing
- RBAC: system roles (user / moderator / admin) + contextual roles (client / freelancer)
- Audit log для всіх критичних подій
